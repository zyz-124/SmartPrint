在项目目录 subjects/{subject}/ 下创建 {topic}.json，格式如下：

{
  "subject": "{subject}",
  "center": "{topic}",
  "subtitle": "（课本版本信息，如 人教版必修二 第4单元）",
  "branches": [
    {"title": "一级标题（课本小节名）", "points": [
      "二级要点：该节第一个重要知识点的完整描述",
      "二级要点：第二个知识点，包含关键信息和细节",
      "二级要点：第三个知识点，注意与其他知识的关联",
      "二级要点：第四个知识点，补充或拓展内容"
    ]}
  ],
  "timeline": [],
  "key_concepts": [
    {"term": "核心术语", "explain": "课本定义或标准解释"}
  ],
  "key_figures": [
    {"name": "人物姓名", "title": "身份头衔", "contribution": "主要贡献或成就"}
  ],
  "marginalia": [
    "课本旁注、知识拓展、易混辨析等补充内容"
  ]
}

规则：
- 8～12个一级标题，严格按课本小节顺序排列
- 每个一级标题下3～6个二级要点
- 每个二级要点20～35个字，完整表述一个知识碎片
- 层次清晰，上下级逻辑关系明确
- key_concepts：本单元核心术语和定义，6～10个
- key_figures：涉及的重要人物，3～5个
- marginalia：课本边栏、小字、注释内容，4～6条
- 时间线按需填写
- 内容准确，严格依照课本表述
- 只输出 JSON，不要解释
